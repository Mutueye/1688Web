17.
app.js 页面框架通用代码，包含页面响应式控制、浏览器信息检测、导航菜单控制等功能。app.js中包含了device.js和fastclick.js两个插件。 其中device.js插件用于设备检测，从而针对不同的设备（桌面pc/平板/手机)进行响应式控制，device.js会在页面的html标签上添加类似'desktop'、'tablet'、'mobile'等class来区分不同的设备，所以实际项目中需要注意保证html的class属性不被其他插件给清空或更改。fastclick.js解决触摸设备浏览器点击延迟300ms。 详见js/app.js源码。
